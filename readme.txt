WordPress Theme MedicalHospitalWebsite

Тема WordPress сверстана на основе HTML5 шаблона MedicalHospitalWebsite https://github.com/frontend-coder/MedicalHospitalWebsite

Тема состоят из:

главной страницы сайта в виде Landing Pages
внутренних страниц на основе базового шаблона страницы
шаблона постов блога
шаблона страниц докторов, который позволяет предоставлять информацию о работающих врачах, исходя из профессий  
шаблона портфолио, который позволяет размещать на отдельной странице набор фотографий, которые позволяют предоставлять визуальную информацию о деятельности учреждения
шаблона страницы контактов, с помощью которой предоставляется потенциальному клиенту всю контактную информацию
